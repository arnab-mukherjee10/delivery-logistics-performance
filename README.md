## Delivery Logistics Performance Analysis
 ### Problem Statement:
  The main aim of the project is to analyze and optimize delivery logistics performance by integrating data from multiple operational sources
  and transforming it into a unified analytical dataset. It focuses on identifying cost drivers, monitoring delivery delays, and evaluating
  profitability across customers, routes, and drivers. The project leverages SQL and Power BI to build interactive dashboards that provide
  actionable insights and support data-driven decision-making.

 ### Objective:
  •  Perform end-to-end analysis of logistics operations from data cleaning to dashboard creation.
  
  •  Identify insights related to cost efficiency, delivery performance, and profitability.
  
  •  Enable interactive exploration of logistics data for better understanding of operational trends.
  
  •  Support data-driven decision-making through clear and impactful visualizations.
  
 ### Tools and Technologies:
  •  Excel → Data cleaning and preprocessing.
  
  •  PostgreSQL (SQL) → Data integration, transformation, and analysis.
  
  •  Power BI → Data visualization and dashboard creation.

 ### Data Workflow:
   ### 1.) Data Cleaning and Preparation (Excel):
  •  Handled missing values and removed duplicates.
  
  •  Standardized column formats and data types.
  
  •  Cleaned categorical fields (e.g., delivery status, locations).
  
  •  Prepared structured datasets for further processing.

   ### 2.) Data Integration and Transformation (SQL):
  •  Combined multiple datasets like trips, loads, customers, etc using joins to create a unified dataset.
  
  •  Derived new fields such as profit, average delay and fuel cost per mile.
  
  •  Performed analytical queries to uncover key insights:
  
  i) Profit Per Customer
  
  ii) High Delay Routes
  
  iii) Fuel Cost Efficiency
  
  iv) Driver Performance
  
  v) Monthly Revenue and Profit Trend

   ### 3.) Data Modeling and Visualization (Power BI):
  Built a structured data model with relationships.
  ### Key KPIs:
  •  Total Revenue
  
  •  Total Profit
  
  •  Transportation Cost %
  
  •  Cost Per Mile
  
  •  On-Time Delivery %
  
  ### Key Visuals:
  •  Top Customers by Profit
  
  •  Yearly Profit and Revenue Trends
  
  •  Revenue by Route
  
  •  Delay Categories
  
  •  Cost vs Revenue

 ### Key Insights:
  •  Identified high cost and high delay routes.
  
  •  Highlighted top-performing customers and drivers.
  
  •  Analyzed delay patterns and delivery performance.
  
  •  Evaluated cost-to-revenue efficiency trends over time.

 ### Business Impact:
  •  Helps stakeholders monitor logistics performance.
  
  •  Enables identification of operational inefficiencies.
  
  •  Supports data-driven decisions to reduce cost and improve delivery performance.
